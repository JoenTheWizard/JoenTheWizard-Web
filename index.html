WELCOME TO MVCMD

What is mVCMD?

The Multivariable Command Line (abbreviated as mVCMD) is a multi-functional CLI program created in C# and Python. The aim of mVCMD is to create as many functional tools all compacted into one command-line program. This ranging from the default command-line interpreter, cyber-security/pen-testing to even solving and plotting mathematical equations. It will also include its own compiler later in the future.
